# Fork

**توضیح:**  
Fork یک نسخه کپی از یک repository دیگر روی حساب GitHub شماست.  

**کاربرد:**  
- وقتی می‌خوای روی پروژه‌ی کسی کار کنی بدون اینکه تغییراتت روی repo اصلی اعمال شود.  
- پس از fork، می‌توانی تغییراتت را محلی یا آنلاین ایجاد کرده و سپس Pull Request ارسال کنی.  

**منابع:**  
- [GitHub Docs: Fork a repo](https://docs.github.com/en/get-started/quickstart/fork-a-repo)
