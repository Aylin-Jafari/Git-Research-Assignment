# تفاوت Pull, Pull Request و Push

| دستور | توضیح |
|-------|-------|
| pull  | گرفتن آخرین تغییرات از repo آنلاین و ادغام با نسخه محلی |
| push  | ارسال تغییرات محلی به repository آنلاین |
| Pull Request | درخواست برای ادغام تغییرات شما با repo اصلی و بررسی توسط مالک |

**نکته:**  
- Pull تغییرات را مستقیم اعمال می‌کند  
- Push تغییرات محلی را به repo آنلاین می‌فرستد  
- PR برای همکاری و بررسی قبل از merge است
