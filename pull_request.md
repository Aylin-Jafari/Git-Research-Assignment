# Pull Request (PR)

**توضیح:**  
Pull Request درخواست برای ادغام تغییرات شما با repository اصلی است.  

**کاربرد:**  
- وقتی روی fork یا branch تغییرات داده‌اید  
- مالک اصلی repository تغییرات را بررسی و merge می‌کند  

**منابع:**  
- [GitHub Docs: About Pull Requests](https://docs.github.com/en/pull-requests)
